# Sive wave story

Данное приложение является формой, которая строит визуализацию синусоидального сигнала.
Поддерживается работа с несколькими потоками.

## Технологии 

- [.NET CORE 3.1](https://dotnet.microsoft.com/en-us/download/dotnet/3.1)
- [ScottPlot](https://scottplot.net)
- [WPF](https://learn.microsoft.com/ru-ru/dotnet/desktop/wpf/get-started/create-app-visual-studio?view=netdesktop-7.0)

## Запуск

``cd SineWaveStory.App``

``dotnet run``
